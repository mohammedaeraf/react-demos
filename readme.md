## 🚧 Getting Started

### ✅ Requirements

- Node.js ≥ 14.x
- npm ≥ 6.x or yarn
- React ≥ 18.x
- A code editor like **VS Code**

### 📦 Installation

```bash
git clone https://github.com/mohammedaeraf/react-demos.git
cd react-demos
npm install
npm run dev   # for Vite
# or
npm start     # for Create React App
```
